# Utility classes

Utility classes enables you to create your own utilites to meet your own needs in your project.

## 1. Add classes to theme file

```javascript
const theme = {
  // all of your theme properties...
  classes: {
    'heading-1': {
      fontSize: 16,
      color: 'dodgerblue',
    },
  },
}
```

## 2. Use your class

```javascript
<Text class="heading-1">This is a heading</Text>
```
